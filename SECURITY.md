# Security Policy

## Reporting Security Issues

If you discover a security vulnerability:

1. **DO NOT** open a public issue
2. Email **security@merlionclaw.sg** with details
3. Include steps to reproduce if possible
4. Allow 72 hours for response

## Scope

This security policy covers:
- The MerlionClaw website
- Any code in this repository
- Community-submitted skills

## What We Consider Security Issues

- XSS vulnerabilities
- CSRF vulnerabilities
- Data exposure
- Malicious skill submissions

## Response Time

- Acknowledgment: 24 hours
- Initial assessment: 72 hours
- Resolution: Depends on severity

## Safe Harbor

We support responsible disclosure and will not pursue legal action against researchers who:
- Follow this policy
- Do not exploit vulnerabilities beyond proof of concept
- Do not access others' data

# AgentHub — AI Agent Marketplace

**Status:** In Development (Started April 8, 2026)  
**Launch Target:** Beta (May 2026), Public (June 2026)  
**Model:** 30% take rate on all transactions  

---

## 🎯 What We're Building

**Marketplace for autonomous AI agents:**
- Developers list agents (skills, services, tools)
- Customers browse, subscribe, deploy
- We handle billing, delivery, support infrastructure
- 30% to AgentHub, 70% to developer

## 💰 Revenue Model

| Transaction Type | Price | Our Cut (30%) | Developer (70%) |

| One-Time Purchase | $5,000 | $1,500 | $3,500 |

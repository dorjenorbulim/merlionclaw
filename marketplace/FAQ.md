## 🚀 General Questions

### What is AgentHub?

AgentHub is a marketplace for autonomous AI agents. Think "App Store for business AI."

- **Developers** list AI agents (skills, services, tools)
- **Customers** browse, subscribe, and deploy in minutes
- **We handle** billing, distribution, and support infrastructure
- **Revenue split:** 70% to developers, 30% to AgentHub

### What's an AI Agent?

An AI agent is an autonomous program that:
- Takes actions without constant human input
- Makes decisions based on rules or AI models
- Completes tasks end-to-end (not just answering questions)

**Examples:**
- Compliance checker (screens transactions for AML/CFT)
- Research bot (monitors news, generates summaries)
- Trading bot (executes trades based on signals)
- Customer support agent (handles tickets autonomously)

### What's the Difference Between AgentHub and Other AI Platforms?

| Platform | What It Does | AgentHub Difference |
|----------|--------------|---------------------|
| **ChatGPT** | Conversational AI | We host autonomous agents (not just chat) |
| **LangChain** | Developer framework | We provide marketplace + distribution |
| **Zapier** | Workflow automation | We focus on AI agents (not just if-this-then-that) |
| **RapidAPI** | API marketplace | We host full agents (not just API endpoints) |

## 🔒 Security & Privacy

### Is My Data Secure?

**Yes.** We use industry-standard security:

- **Encryption:** HTTPS for all traffic, AES-256 for data at rest
- **Access Controls:** Only authorized personnel can access data
- **Password Security:** Hashed (bcrypt), never plain text
- **Regular Audits:** Security reviews every 6 months

**During Beta:** Some processes are manual (founder may access data for support). Still protected, just less automated.

### What Data Do You Collect?

**From All Users:**
- Account info (name, email, password hash)
- Usage data (which agents, how often)
- Device info (browser, OS, IP address)

**From Customers:**
- Payment info (processed by Stripe, not stored by us)
- Agent input/output (varies by agent)

**From Developers:**
- Agent code and documentation
- Revenue and payout info
- Bank account (for payouts)

**See full details:** [Privacy Policy](/legal/privacy-policy.md)

### Do You Sell My Data?

**No.** We do not and will not sell your personal data.

**We share data only when necessary:**
- With developers (anonymized usage of their agents)
- With service providers (Stripe, AWS — they help us operate)
- With legal authorities (if required by law)

### Where Is Data Stored?

**Primary:** Singapore (AWS servers)  
**Payment Processing:** Stripe (US-based, PCI compliant)  
**Backups:** Singapore (encrypted)

**Cross-border transfers:** Protected via standard contractual clauses.

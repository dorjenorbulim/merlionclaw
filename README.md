## About

MerlionClaw is an independent community project by and for Singapore OpenClaw users. We showcase ethical use cases, share custom skills, and help each other use AI to improve our lives.

## 🛡️ Ethical Framework

All submissions must meet these standards:

- ✅ **Transparency** — Clear about what the AI does
- ✅ **Human Agency** — Humans remain in control
- ✅ **Privacy First** — Respect personal data (PDPA compliant)
- ✅ **Social Good** — Enriches lives, not just wallets
- ✅ **Legal Compliance** — Follows Singapore laws
- ✅ **No Harm** — Doesn't deceive or exploit

## 🤝 Contributing

### Submit a Story

1. Fork this repository
2. Create a new story in `/stories/`
3. Use the template from `stories/hawker-automation.html`
4. Submit a PR with clear description

### Submit a Skill

1. Fork this repository
2. Add your skill to `/skills/index.html`
3. Follow the existing card format
4. Submit a PR

### Content Guidelines

- Be honest about your experience
- Include concrete outcomes when possible
- Respect privacy (yours and others')
- Disclose any commercial relationships

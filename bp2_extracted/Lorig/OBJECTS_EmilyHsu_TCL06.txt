Objects
zObject : that which is known by an 
awareness
zThree divisions:
1.Appearing object 
2.Determined object (for conceptual consciousnesses only)
3.Object of engagement 
1) Appearing object (snang yul)
zIn general, the appearing object for an awar eness is the object  that appears to 
the particular perception or conception. 
zThe appearing object of a particular awareness and apprehended object of that 
awareness are synonymous /mutually inclusive
zWhatever is an established base is necessarily an appe aring object. This is 
because (1) whatever is a (functioning)  thing is necessarily the appearing 
object of a direct perceiver, and
z(2) whatever is permanent is necessarily the appearing object of a conceptual 
consciousness.
zThe first reason is established because appear ing object of a direct perceiver, 
apprehended object [of a direct percei ver], and thing are synonymous. 
zThe second reason is established becaus e appearing object of a conceptual 
consciousness, apprehended object of a conceptual consciousness, and 
permanent phenomenon are synonymous.
2) Determined object (zhen yul)
(also sometimes translated as “conceived object”)
The determined object is for conceptual 
consciousnesses only. 
zDetermined object of thought, 
zObject of engagement of thought, and 
zObject of the mode of apprehension of 
thought
are synonymous.
3) Object of engagement (‘jug yul)
The object of engagement is the main object of 
the awareness.
zObject of engagement of a direct perceiver 
and
zObject of the mode of apprehension of a 
direct perceiver
are synonymous/mutually inclusive.
Object of the mode of apprehension
zWhatever is an established base is necessarily the 
object of the mode of apprehension of both a conceptual and a non-conceptual consciousness.
zWhatever is the object of the mode of apprehension of the thought consciousness apprehending it is not 
necessarily an object of the mode of apprehension of thought. 
–e.g., horns of a rabbit are not an object of the mode of 
apprehension of thought.
Three types of objects – Summary 
1.Appearing object, apprehended object
2.Determined object (or conceived object)
3.Object of engagement, object of the mode of apprehension
zFor direct perceivers:
–appearing object of a direct perceiver, apprehended object of a direct 
perceiver, and functioning thing are synonymous
–object of engagement an d object of the mode of apprehension are 
synonymous
zFor conceptual consciousnesses: 
–appearing object of a conceptual c onsciousness, apprehended object of a 
conceptual consciousness, and per manent phenomenon are synonymous
–determined object, object of engagemen t, and the object of the mode of 
apprehension are synonymous.

## 📋 Executive Summary

**PainPoints.sg** is a Singapore-centric community platform where residents post everyday problems, get help from verified experts and community members, and discover trusted solutions — all free for users, funded by B2B sponsors who gain insights and brand goodwill.

**Mission:** Reduce suffering, one solved problem at a time.

**Vision:** Become Singapore's most trusted platform for everyday problem-solving — where no Singaporean suffers alone.

**Why Now:**
- Singaporeans face unique challenges (HDB rules, COE bidding, CPF withdrawals, NS obligations, healthcare costs, school registration, etc.)
- Existing platforms (Reddit, Quora, HardwareZone) are global or unstructured — no Singapore-specific focus
- No platform connects problem-posters with verified local experts + government resources
- Companies desperately need real-time insights into Singaporean pain points (product teams, startups, government agencies)

**Business Model:** Sponsorship-first (B2B), free for users (B2C)

**Year 1 Goals:**
- 50,000 registered users
- 10,000 pain points posted
- 60% resolution rate
- 20 corporate sponsors
- $500K revenue (100% from sponsors)

**Funding Sought:** $150K seed (for development, marketing, operations) — OR bootstrap with sponsor pre-sales

### Current Solutions (All Flawed):

| Platform | Problem |
|----------|---------|
| **Reddit r/askSingapore** | Unverified advice, global users, no expert verification |
| **HardwareZone Forums** | Toxic culture, hard to search, no resolution tracking |
| **Quora** | Global (not SG-specific), outdated answers, no local experts |
| **Google Search** | Scattered results, hard to distinguish credible sources |
| **Government Websites** | Fragmented across 50+ agencies, hard to navigate |
| **Facebook Groups** | Closed ecosystems, advice disappears in feed |

### The Gap:
**No Singapore-specific platform that:**
- ✅ Aggregates ALL everyday pain points in one place
- ✅ Connects posters with verified local experts
- ✅ Tracks resolution status (Open → Solved)
- ✅ Links to official government resources
- ✅ Rewards helpers for quality advice
- ✅ Provides real-time insights to companies & government

## 🎯 Solution Overview

### Platform Features (Phase 1 — MVP):

#### For Users (Free):
| Feature | Description |
|---------|-------------|
| **Post Pain Point** | Text, image, or voice note (anonymous or identified) |
| **Categorize** | HDB, COE, CPF, NS, Healthcare, Education, Employment, Family, Consumer, etc. |
| **Upvote** | "Me too" button indicates shared pain point |
| **Browse** | Search by category, urgency, recency, resolution status |
| **Comment** | Provide advice, ask follow-up questions |
| **Mark Solved** | Original poster marks best answer as "Solved" |
| **Rate Helper** | 1-5 star rating for quality of advice |
| **Resource Links** | Auto-suggest government links (e.g., HDB, CPF Board, MOM) |

#### For Helpers/Experts (Free, with verification):
| Feature | Description |
|---------|-------------|
| **Browse Pain Points** | Filter by category, urgency, bounty amount |
| **Provide Advice** | Text, video, or voice response |
| **Build Reputation** | Rating score, badges, levels (Novice → Expert → Master) |
| **Verified Badge** | Professionals with credentials (lawyer, doctor, financial advisor) |
| **Profile Page** | Showcase expertise, ratings, testimonials |
| **Direct Messages** | For follow-up consultations (free or paid) |

#### For Sponsors (Paid):
| Feature | Description |
|---------|-------------|
| **Sponsored Categories** | "This category sponsored by DBS" (e.g., Finance) |
| **Branded Challenges** | "Singtel Pain Point Challenge: Telco Issues" (prize pool + brand visibility) |
| **Insights Dashboard** | Real-time pain point trends, sentiment analysis, category volumes |
| **Custom Reports** | Quarterly "Top 100 Pain Points in Singapore" report |
| **Expert Network Access** | Connect with verified experts for partnerships |
| **CSR Alignment** | Brand associated with helping Singaporeans (goodwill) |

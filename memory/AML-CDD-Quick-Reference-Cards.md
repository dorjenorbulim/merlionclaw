# AML/CDD Quick Reference Cards
## Print-Ready Design (5 Cards)

**Format:** A4 sheet, 5 cards per page (cut along lines)  
**Size:** Each card ≈ 9cm x 14cm (credit card size x 2)  
**Paper:** Cardstock recommended (160–200 gsm)  
**Print:** Double-sided (optional: laminate for desk reference)  
**Quantity:** 50+ participants (print 10+ sheets for extras)

---

## 🖨️ Print Instructions

1. **Open this document** in any markdown viewer
2. **Copy each card** into a design tool (Canva, PowerPoint, Illustrator)
3. **Arrange 5 cards per A4 page** (or print individually)
4. **Print on cardstock** (160–200 gsm for durability)
5. **Cut along dotted lines**
6. **Optional:** Laminate for long-term desk reference
7. **Distribute:** One set per participant (keep at desk)

## CARD 2: The 5 CDD Elements

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  THE 5 CDD ELEMENTS                                             │
│  What Information Must You Collect?                             │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 1. Customer Identity                                      │ │
│  │    • Name, registration number, registered address        │ │
│  │    • Individuals: NRIC/passport, DOB, nationality         │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 2. Beneficial Ownership (BO)                              │ │
│  │    • Who ultimately owns or controls?                     │ │
│  │    • Typically 25%+ ownership or significant control      │ │
│  │    • Must identify HUMANS, not companies                  │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 3. Purpose & Nature of Business                           │ │
│  │    • What does this company actually do?                  │ │
│  │    • Expected transaction patterns                        │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 4. Source of Wealth (SoW)                                 │ │
│  │    • How did they accumulate their wealth?                │ │
│  │    • Business profits, inheritance, investments           │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 5. Source of Funds (SoF)                                  │ │
│  │    • Where is the money for THIS transaction from?        │ │
│  │    • Bank account, sale of asset, loan                    │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  REMEMBER: SoW = how they got rich.                           │
│            SoF = where this money came from.                  │
│            Different questions. Both matter.                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## CARD 5: Documentation Golden Rules

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  DOCUMENTATION GOLDEN RULES                                     │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 1. If it's not documented, it didn't happen.              │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 2. Document in real-time (not from memory later).         │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 3. Be specific.                                           │ │
│  │    Not: "Documents verified"                              │ │
│  │    But: "NRIC S1234567A verified via original sighted     │ │
│  │         on 15 Mar 2026"                                   │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 4. Include your name/ID and date on all entries.          │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 5. Retain records for minimum 5 years.                    │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 6. If you escalate, document the escalation.              │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 7. If customer refuses, document the refusal.             │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ 8. If something feels off, document why.                  │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  REMEMBER: MAS examiners look at the system.                  │
│            Not your memory. Not your notes. The system.       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 🎨 Design Specifications

### Card Dimensions
- **Width:** 90mm (9cm)
- **Height:** 140mm (14cm)
- **Bleed:** 3mm on all sides (for professional printing)
- **Safe Zone:** 5mm from edges (keep text inside this area)

### Typography
- **Card Title:** 14pt, Bold, Primary Blue (#003366)
- **Section Headers:** 11pt, Bold, Secondary Blue (#0052A3)
- **Body Text:** 9pt, Regular, Neutral Grey (#757575)
- **Reminder Boxes:** 9pt, Bold, White text on colored background

### Colors
| Element | Color | Hex Code |
|---------|-------|----------|
| Card Title | Primary Blue | #003366 |
| Section Headers | Secondary Blue | #0052A3 |
| Body Text | Neutral Grey | #757575 |
| Low Risk | Green | #388E3C |
| Medium Risk | Yellow | #FBC02D |
| High Risk | Red | #D32F2F |
| Reminder Boxes | Blue | #0052A3 |
| Background | White | #FFFFFF |

### Icons
- Use flat, minimalist icons
- Size: 16x16px or 24x24px
- Color: Match section header (Secondary Blue)
- Style: Consistent across all cards

### Layout
- 5 cards per A4 page (2 columns: 3 cards left, 2 cards right)
- Dotted cut lines between cards (0.5pt, grey)
- Crop marks at corners (for professional cutting)
- Cardstock: 160–200 gsm (recommended for durability)

# AML/CDD Quiz — Google Forms Setup Guide
## Digital, Auto-Graded Assessment

**Platform:** Google Forms (free, integrates with Google Workspace)  
**Alternative:** Typeform (paid, more polished)  
**Quiz:** 10 questions, 80% pass mark (8/10), auto-graded  
**Delivery:** Hybrid (in-room via QR code + Zoom link for virtual)  
**Completion Time:** 5 minutes  

---

## 📋 Step-by-Step Setup (Google Forms)

### Step 1: Create New Form (5 minutes)

1. Go to **forms.google.com**
2. Click **"+ Blank"** (or use template gallery)
3. Form title: **"AML/CDD Training Quiz — MAS Notice 626"**
4. Form description:
   ```
   Knowledge Check Quiz for Frontline Sales Managers (Corporate Banking)
   
   • 10 questions (multiple choice + true/false)
   • Pass mark: 80% (8/10 correct)
   • Time limit: 5 minutes
   • Open book (you can refer to your notes)
   • Certificate issued upon passing
   
   If you don't pass: Retake available after 24 hours. Maximum 3 attempts.
   ```

---

### Step 2: Enable Quiz Mode (2 minutes)

1. Click **Settings** (gear icon, top right)
2. Scroll to **"Make this a quiz"**
3. Toggle **ON**
4. Settings:
   - **Release grade:** Immediately after submission
   - **Respondents can see:** Missed questions, Correct answers, Point values
   - **Respondents can:** [ ] Retake quiz (we'll manage this manually)

---

### Step 3: Add Questions (15 minutes)

**For each question below:**
1. Click **"+ Add question"**
2. Select question type (Multiple choice or True/False)
3. Copy question text
4. Add answer options
5. Click **"Answer key"** (bottom of question)
6. Select correct answer(s)
7. Assign points: **1 point per question**
8. Click **"Done"**

## 📊 Response Management

### View Responses (During/After Session)

1. Click **"Responses"** tab (top of form)
2. **Summary:** See aggregate scores, question-by-question analysis
3. **Individual:** See each participant's responses
4. **Export to Sheets:** Click green Sheets icon (creates spreadsheet)

### Grading & Certification Workflow

**Immediately After Session:**
1. Open Google Forms → Responses
2. Export to Google Sheets (if not auto-created)
3. Filter by score:
   - **Pass (8–10):** Add to certificate list
   - **Fail (<8):** Add to retake list

**Within 24 Hours:**
1. **For passes:**
   - Generate certificates (use template below)
   - Email certificates with subject: "AML/CDD Training Certificate — [Name]"
   
2. **For fails:**
   - Email retake link with subject: "AML/CDD Quiz Retake — [Name]"
   - Include: "Your score: X/10. Pass mark: 8/10. Retake available after 24 hours. Maximum 3 attempts."

**Within 48 Hours:**
1. **For 3rd failures:**
   - Email: "1:1 Coaching Session Required"
   - Schedule individual session (30 minutes)
   - Document coaching in training records

## 🛠️ Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Form won't load | Check internet connection. Try incognito window. |
| Responses not saving | Ensure "Required" toggle is ON for all questions. |
| Score not displaying | Check quiz mode is enabled in Settings. |
| Email not sent | Check Gmail outbox. Verify email address is correct. |
| QR code not scanning | Increase QR code size. Use high-contrast colors. |
| Participant can't retake | Check "Limit to 1 response" is OFF in Settings. |
| Certificate not auto-generated | Check Autocrat setup. Verify column headers match. |
